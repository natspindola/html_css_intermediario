O HTML também possui Pré Processadores, além de tentar facilitar a criação de HTML, estes Pré Processadores também tentam fazer com que o nosso código seja mais limpo, sem repetições e fácil de ler. Por isso eles normalmente possuem uma sintaxe bem diferente do HTML. No fim teremos HTML, já que é ele que o navegador consegue interpretar.

Exemplo de código no PUG:

div#minhaDiv.classe1.classe2 Meu Conteúdo
ul
li
li
input(
type='checkbox'
name='agreement'
checked
)
ul
each numero in [1, 2, 3, 4, 5]
li= numero
- var isOpen = true
div
if isOpen
span Aberto
else
span Fechado

Como ficaria no HTML:

< div class="classe1 classe2" id="minhaDiv">Meu Conteúdo
< ul>
< li>
< li>

< input type="checkbox" name="agreement" checked="checked" />
< ul>
< li>1
< li>2
< li>3
< li>4
< li>5

< div>< span>Aberto
