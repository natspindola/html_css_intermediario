O SVG (Scalable Vector Graphics) é uma linguagem XML que utilizamos para a criação de imagens vetoriais.

BITMAP

 As imagens com as quais costumamos trabalhar ( do tipo PNG, JPEG, GIF, etc ) são conhecidas como Imagens Bitmap. Isso porque são basicamente um mapa de bits. Essas imagens são compostas de vários pixels coloridos, pequenos quadradinhos que juntos formam uma imagem.

VETORIAIS

 Diferente das imagens bitmap, as imagens vetoriais não são compostas por pixels. Elas são compostas por fórmulas matemáticas que descrevem formas. Isso significa que as imagens vetoriais nunca perdem qualidade, não importa se deixarmos a imagem do tamanho da cabeça de um alfinete ou do tamanho de um grande prédio.

Vantagens e desvantagens

Bitmap

Vantagens Desvantagens
Bom para fotografias por capturar cada cor de cada ponto de uma imagem que, ao ser apresentado em um monitor dão a sensação de uma imagem mais realista, transmitindo melhor a mudança de tons, cores e sombras. Ao ampliar a imagem, o pixel fica maior, fazendo a imagem perder qualidade.
Pouco processamento para leitura, pois cada pixel só é apresentado; Arquivos mais pesados, pois cada pixel deve ter as informações de cor e posição em que deve ser apresentado;

Vetores

Vantagens Desvantagens
Ao redimensionar a imagem não se perde qualidade. Não dá para ter todos os detalhes de cada ponto que seriam necessários para algo como uma fotografia.
Arquivos menores, pois as formas são identificadas por fórmulas matemáticas; Precisa de mais processamento para leitura, já que são feitos cálculos para poder apresentar as formas;
